# WalkApp — a Go + SQLite + React training guide

A start-from-scratch course that teaches Go, SQLite, and React by building one real,
useful thing: a tool for cataloguing walking paths by what they're good for (shade,
garbage cans, loops, parking, dog-friendliness) that then suggests where to walk today
based on your mood, the weather, and your criteria.

No prior experience with any of the three is assumed. Everything you build accrues to the
app — there are no throwaway exercises.

## Read it

- **Web page:** https://egon42.github.io/walkapp-guide/
- **Plain text:** [`LESSON-PLAN.md`](LESSON-PLAN.md) (also renders on GitHub)

The web version (`index.html`) is a single self-contained file — no dependencies, no build
step. It's designed to be skimmed: the colored callouts carry the details that matter.
